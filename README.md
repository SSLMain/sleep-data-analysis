# Sleep Data Analysis

This is a beginner Python project analyzing sleep data.

## Tools
Python  
pandas  
matplotlib  

## Features
Loads sleep data from a CSV file  
Calculates average sleep hours  
Visualizes sleep hours per person  
Analyzes caffeine intake vs sleep  

## Files
sleep_analysis.py – main analysis script  
sleep_data.csv – dataset used for analysis  
sleep_chart.png – generated visualization
